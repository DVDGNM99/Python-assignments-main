ğŸ”¹ Overview

Chem-Reporter is a Python desktop application using Tkinter for chemical data retrieval and reporting.
It fetches chemical information from PubChem (via PUG-REST + PUG-View) starting from a SMILES string or a CSV list of molecules.
The project is modular and extensible, supporting GUI, batch mode, and structured result folders.
