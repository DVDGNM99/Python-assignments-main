# Snake Game using Pygame

This project provides a simple Python implementation of the classic
Snake game using the Pygame library.

# a. Requirements

-   Python 3.11 (or a compatible version)
-   Conda (optional, if you want to use the provided environment.yml)
-   Pygame library

# b. Setup Instructions

1.  Create the Conda environment from the environment.yml file: conda
    env create -f environment.yml

2. go to the directory-> cd "DIRECTORY"

3.  Activate the environment: conda activate game

4.  Run the game script: C:\Python-assignments-main\day05> python snake.py

# c. Gameplay

-   Use the arrow keys to move the snake (Up, Down, Left, Right).
-   Eat the red food blocks to grow.
-   Avoid hitting the walls or your own tail.
-   When a collision happens, the game closes automatically.
